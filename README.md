# <img src="https://cdn4.iconfinder.com/data/icons/iconsimple-logotypes/512/github-256.png" width="40px" height="40px"/> Github - RoBorregos Workshop

Bienvenido a al entrenamiento de RoBorregos sobre Github. Las guías en este repositorio pretenden enseñar los conceptos básicos necesarios para trabajar con la herramienta de Github, guiar paso a paso a nuevos usuarios de Github para su instalación y configuración, mostrar uno de los estándares de trabajo más utilizados a nivel profesional, realizar un ejercicio como ejemplo y finalmente establecer los estándares que se seguirán en la organización de RoBorregos en Github.
